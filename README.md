# music API

این پروژه یک API برای مدیریت موسیقی‌های سبک رپ است که با استفاده از معماری Clean Architecture و اصول Clean Code توسعه یافته است.

## ویژگی‌های پروژه 🎤

- **هنرمندان (Artists)**: مدیریت اطلاعات هنرمندان شامل نام و عکس پروفایل.
- **آلبوم‌ها (Albums)**: مدیریت آلبوم‌های موسیقی شامل نام و کاور.
- **موسیقی‌ها (Musics)**: مدیریت اطلاعات موسیقی‌ها.
- **لایک‌ها (Likes)**: سیستم لایک برای موسیقی‌ها و آلبوم‌ها.
- **کامنت‌ها (Comments)**: امکان اضافه کردن و مدیریت کامنت‌ها.
- **پلی‌لیست‌ها (Playlists)**: مدیریت پلی‌لیست‌های عمومی و خصوصی.

## تکنولوژی‌های مورد استفاده 🛠️

- **FastAPI**: فریم‌ورک مدرن برای ساخت API‌های سریع و مقیاس‌پذیر.
- **SQLAlchemy**: ORM برای تعامل با پایگاه‌داده‌ها به صورت غیرهمزمان.
- **PostgreSQL**: پایگاه‌داده رابطه‌ای برای ذخیره‌سازی داده‌ها.
- **Pydantic**: برای اعتبارسنجی داده‌ها.
- **JWT (JSON Web Tokens)**: برای احراز هویت و مدیریت جلسات.
- **Google OAuth**: برای ثبت‌نام و ورود با حساب Google.


## طراحی غیرهمزمان (Async) ⚡

پروژه به صورت **غیرهمزمان (Async)** توسعه یافته است تا عملکرد بهینه‌تری داشته باشد و به خوبی با حجم بالای درخواست‌ها مدیریت شود.

## پیشرفت پروژه 🚧

این پروژه هنوز در حال توسعه است و به زودی ویژگی جستجو (Search) اضافه خواهد شد. به محض تکمیل این ویژگی، آن را در کامیت‌های بعدی منتشر خواهیم کرد.

## نصب و راه‌اندازی ⚙️

1. **کلون کردن پروژه**:
    ```bash
    git clone https://github.com/Soli1212/RapAPI.git
    ```

2. **ورود به پوشه پروژه**:
    ```bash
    cd RapAPI
    ```

3. **نصب وابستگی‌ها**:
    ```bash
    pip install -r requirements.txt
    ```

4. **تنظیم متغیرهای محیطی**: فایل `.env` را با تنظیمات پایگاه‌داده و دیگر نیازها پیکربندی کنید.

5. **راه‌اندازی سرور**:
    ```bash
    python main.py
    ```

## استفاده 📡

برای استفاده از API، از ابزارهایی مانند Postman یا Curl برای ارسال درخواست‌ها و دریافت پاسخ‌ها استفاده کنید. ثبت‌نام و ورود از طریق Google OAuth انجام می‌شود.
